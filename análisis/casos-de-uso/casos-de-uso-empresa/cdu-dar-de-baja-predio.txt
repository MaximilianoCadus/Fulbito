Caso de uso: Dar de baja predio

Actores:
- Usuario empresa (Primario)

Pre-condiciones:
- Se debe tener acceso a internet
- Se debe tener una sesión iniciada
- Se debe tener un predio dado de alta

Camino básico:
1- El usuario ingresa a la sección de predios
2- El sistema muestra los predios
3- El usuario selecciona un predio
4- El sistema muestra los datos del predio
5- El usuario selecciona la opción de dar de baja predio
6- El sistema valida la acción y muestra mensaje de éxito

Caminos alternativos:
6.1- El predio no puede ser dado de baja
6.1.1- El sistema no logra validar la acción y muestra mensaje de error

Escenario de fracaso:
- El usuario no logra dar de baja el predio

Escenario de éxito:
- El usuario logra dar de baja el predio
