Caso de uso: Cambiar datos de perfil (Usuario/Empresa)

Actores:
- Usuario/Empresa (Primario)

Pre-condiciones:
- Se debe tener acceso a internet
- Se debe tener una sesión iniciada

Camino básico:
1- El usuario ingresa a la sección de datos de perfil
2- El sistema muestra los datos del perfil
3- El usuario selecciona el dato a cambiar y lo cambia
4- El sistema valida el dato nuevo y muestra mensaje de éxito

Caminos alternativos:
4.1.1 El sistema no logra validar el datao nuevo y muestra mensaje de error

Escenario de fracaso:
- El usuario no logra cambiar el dato del perfil

Escenario de éxito:
- El usuario logra cambiar el dato del perfil
