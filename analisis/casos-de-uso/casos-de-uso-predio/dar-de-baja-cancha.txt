Caso de uso: Dar de baja cancha

Actores:
- Predio (Primario)

Pre-condiciones:
- Se debe tener acceso a internet
- Se debe tener una sesión iniciada
- Se debe tener una cancha dada de alta

Camino básico:
1- El usuario ingresa a la sección de canchas
2- El sistema muestra las canchas
3- El usuario selecciona una cancha
4- El sistema muestra los datos de la cancha
5- El usuario selecciona la opción de dar de baja cancha
6- El sistema valida la acción y muestra mensaje de éxito

Caminos alternativos:
6.1- La cancha no puede ser dada de baja
6.1.1- El sistema no logra validar la acción y muestra mensaje de error

Escenario de fracaso:
- El usuario no logra dar de baja la cancha

Escenario de éxito:
- El usuario logra dar de baja la cancha
