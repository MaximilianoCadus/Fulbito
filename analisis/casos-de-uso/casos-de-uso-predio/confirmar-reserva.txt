Caso de uso: Confirmar reserva

Actores:
- Predio (Primario)

Pre-condiciones:
- Se debe tener acceso a internet
- Se debe tener una sesión iniciada
- Se debe tener una reserva pendiente con seña paga

Camino básico:
1- El usuario ingresa a la sección de reservas pendientes con seña paga
2- El sistema muestra las reservas
3- El usuario selecciona la reserva que desea confirmar
4- El sistema muestra los datos de la reserva seleccionada
5- El usuario confirma el comprobante de pago de la seña de la reserva pendiente
6- El sistema confirma la reserva y muestra mensaje de éxito

Escenario de fracaso:
- El usuario no logra confirmar una reserva pendiente

Escenario de éxito:
- El usuario logra confirmar una reserva pendiente
